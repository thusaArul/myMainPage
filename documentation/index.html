A) HTML Structure - top

This theme has two main content area. Left one is for basic info and contact form. Right section is for Portfolio, awards and experience. Right section is nested within a div with a class of "page-right" and left area with a class of "page-left".

For the main page left section you can see everthing nested within a div with a class of "side-left" and "side-right".

For bottom slide content you can see everything nested within a div with an id of "clientsDashboard".

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

    .page-inner p a {
		color: #someColor;
	} 
    .page-inner p a:hover {
		color: #someColor;
	}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

If you want to change the image on main page i.e. 683x768

	.side-right {
		background: #urlOfYourImage;
	}

To change the right background image when right section is opened.

	.open-right .side-right {
	background: #urlOfYourImage;
	}

C) JavaScript - top

This theme imports three Javascript files.

  1. jQuery
  2. My custom scripts
  3. Some plugin
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Some of the animation in this site is carried out from the customs scripts. The code contains in "common.js".
  3. In addition to the custom scripts, I implement a few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method. For example:

	onAfter:function( anchor, settings ){
		// The 'this' contains the scrolled element (#content)
	}
});
